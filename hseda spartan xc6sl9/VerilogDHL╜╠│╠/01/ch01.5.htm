1.5 C header file listings are normative

The header file listings included in the annexes E, F, and G for veriuser.h , acc_user.h and vpi_user.h are a normative part of the IEEE 1364 Verilog PLI standard. All compliant software tools should use the same function declarations, constant definitions and structure definitions contained in these header file listings.
